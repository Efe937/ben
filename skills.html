Yeteneklerim

Yazılım geliştirme ve problem çözme konularında kendimi sürekli geliştirmekteyim.

HTML5

HTML5, modern web sayfaları için temel yapı taşlarını sunar. Web uygulamaları ve sitelerinin temelini oluşturur.

CSS3

CSS3 ile şık ve duyarlı (responsive) web tasarımları yapabiliyorum. Animasyonlar ve geçişler ile kullanıcı deneyimi sağlıyorum.

JavaScript

JavaScript ile dinamik web sayfaları oluşturuyorum. Asenkron işlemler, API entegrasyonları ve etkileşimli uygulamalar geliştiriyorum.

Python

Python ile güçlü ve esnek Web sistemleri geliştiriyorum. popüler framework'lerle uygulamalar inşa ediyorum.

C#

C# ile masaüstü uygulamaları ve web servisleri (ASP.NET) geliştiriyorum. Modern yazılım çözümleri için güçlü bir dil.

MySQL

Veritabanı yönetim sistemleri arasında MySQL deneyimim var. Web uygulamaları için veritabanı tasarımı ve sorgulama yapabiliyorum.
